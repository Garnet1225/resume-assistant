# resume-assistant

一个用于求职场景的浏览器插件原型，支持结构化《个人简历数据库》：

- 自动识别并填写常见字段（基本信息、联系方式、技能、语言、问答等）
- 支持自定义字段 + 匹配关键词（匹配网页字段时自动填充）
- 支持完整的中国化简历数据库模块（11 大部分）
- 一键记录当前岗位投递，并在弹窗查看进度

## 本地使用

1. 打开 Chrome / Edge 扩展页面 `chrome://extensions`
2. 开启「开发者模式」
3. 选择「加载已解压的扩展程序」并选中本项目目录
4. 打开插件设置页（`编辑简历资料`）填写结构化简历数据库
5. 在招聘网站申请页面点击「自动填表」和「记录本次投递」

## 目录结构

- `manifest.json`：插件清单
- `background/service-worker.js`：状态管理、消息路由、结构化资料归一化
- `content/form-filler.js`：网页表单自动填充逻辑（内置规则 + 自定义字段）
- `popup/`：弹窗 UI（自动填表、投递统计、最近投递）
- `options/`：结构化《个人简历数据库》编辑页

## 数据模块（11 部分）

1. 基本信息
2. 联系方式
3. 教育经历（数组）
4. 工作经历（数组）
5. 项目经历（数组）
6. 专业技能
7. 科研与成果
8. 语言能力
9. 附件材料
10. 常见问答库（中英文 200/500/1000）
11. 岗位匹配模块（方向关键词、自我定位、30 秒陈述）
